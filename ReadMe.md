# AdminKaMU

AdminKaMU is a an early version of an application used to control embedded measuring units(KaMUs).

## Tools

Kaa 0.9.0

Oracle JDK 8

MariaDB 5.5

Zookeeper 3.4.5

Cassandra 3.5

Baasbox 0.9.5

OrientDB 1.7.10

## Versioning
